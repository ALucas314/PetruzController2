# 📚 Documentação do Servidor

Esta pasta contém toda a documentação relacionada ao servidor backend.

## 📖 Guias Disponíveis

### Configuração
- **CONFIGURAR_SUPABASE.md** - Guia completo de configuração do Supabase
- **CONFIGURAR_AGORA.md** - Guia rápido de configuração
- **COMO_ENCONTRAR_API_SUPABASE.md** - Como encontrar as credenciais da API
- **SUPABASE_SETUP.md** - Setup inicial do Supabase

### Troubleshooting
- **TESTAR_CONEXAO.md** - Como testar a conexão com o banco
- **TROUBLESHOOTING.md** - Solução de problemas comuns
- **VERIFICAR_BANCO.md** - Como verificar o banco de dados

### Outros
- **ONDE_ESTA_API_SUPABASE.txt** - Guia rápido de localização da API

## 🚀 Início Rápido

1. Configure o Supabase seguindo **CONFIGURAR_SUPABASE.md**
2. Crie o arquivo `.env` na pasta `server/` com suas credenciais
3. Execute `npm install` e `npm run dev`
4. Teste a conexão seguindo **TESTAR_CONEXAO.md**
